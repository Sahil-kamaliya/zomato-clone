# Zomato Landing Page Clone

This project is a responsive Zomato-inspired landing page built using React.js and Tailwind CSS. The main goal of this project is to create a clean user interface, reusable components, and a smooth frontend experience similar to real-world applications.

## Tech Stack

- React.js  
- Tailwind CSS  
- JavaScript (ES6+)  
- Vite  

## Features

- Responsive design for mobile, tablet, and desktop
- Component-based structure using React
- Clean and modern UI layout
- Reusable components
- Fast development using Vite

## Getting Started

1. Clone the repository:
https://github.com/Sahil-kamaliya/gir-clone

2. Navigate to the project folder:
cd zomato-clone

3. Install dependencies:
npm install

4. Run the development server:
npm run dev
